# SQL Interview Questions Practice

This repository contains **solutions to 50 SQL interview questions** covering commonly used SQL concepts in data analysis.

The problems focus on building strong SQL fundamentals and practicing query patterns that frequently appear in technical interviews for data analyst roles.

---

## Topics Covered

The solutions demonstrate the use of several key SQL concepts, including:

- `GROUP BY` and `HAVING`
- Subqueries
- Common Table Expressions (CTEs)
- Window functions
- Ranking functions (`ROW_NUMBER`, `RANK`, `DENSE_RANK`)
- Analytical functions (`LAG`, `LEAD`)
- Aggregations and filtering
